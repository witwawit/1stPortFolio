Coach Community Website

โปรเจคของเว็บไซต์นี้ มีชื่อว่า Coach Communtity Forums ซึ่งเป็นโปรเจคในการฝึกงานชั้นปีที่ 2 ของบริษัท โกลบิช อคาเดเมีย (Globish Academia) สร้างด้วยภาษา HTML, CSS, JS ผ่านทาง Vue JS Framework, Bootstrap และGithub โดยโปรเจคนี้มีจุดประสงค์สำหรับการตั้งและตอบคำถามของโค้ชสอนภาษาอังกฤษผ่านทางกระทู้ โดยจะมีส่วนที่เป็นแอดมิน(Admin) และส่วนที่เป็นผู้ใช้งาน(Coach) ระบบนี้จะหลายระบบด้วยกันได้แก่ ระบบล็อกอิน สมัครสมาชิก ลืมรหัสผ่าน ยืนยันรหัสผ่านด้วยอีเมล , ระบบการสร้างกระทู้และคอมเมนท์ด้วย Text Editor , ระบบAccount การตั้งชื่อ ตั้งรูปโปรไฟล์ และระบบ Admin ที่จะคอยปักหมุดกระทู้ให้แสดงบนหน้าเว็บไซต์ อนุมัติการสร้างกระทู้ เป็นต้น โดยระยะเวลาในการสร้างโปรเจคใช้เวลาประมาณ 2 เดือน (1 มิถุนายน 2564 - 30 กรกฏาคม 2564)

Moderno Website

Moderno Website เป็นโปรเจคในรายวิชา CMM367 Web Dev II มีจุดประสงค์เพื่อทำให้นักศึกษาทำเว็บไซต์ด้วยภาษา asp.net และใช้ฐานข้อมูลร่วมกับ Microsoft My SQL เป็นหลัก โดยในหน้าเว็บไซต์จะมีการแสดงผลสินค้า มีการกรอกข้อมูลการค้นหาผ่านหน้าเว็บไซต์เพื่อให้แสดงผลสินค้าที่ต้องการ มีการInteractive กับผู้ใช้โดยจะมีการให้ผู้ใช้ ติดต่อมาในส่วนของ Contact ได้หากม่ข้อสงสัย และมีการจัดการหน้าเว็บในส่วนของ Backend โดยระบบจะทำการ ให้ผู้ใช้ Login และตรวจสอบว่าเป็น Admin จริงๆหรือไม่ หากใช่ ระบบก็จะนำผู้ใช้ไปยังส่วนของการจัดการฐานข้อมูลผ่านทางเว็บไซต์ และที่สำคัญยังเป็นเว็บไซต์แบบ Responsive รองรับทุกขนาดหน้าจออีกด้วย

KJ Dreamboys Website

KJ Dreamboys เป็นเว็บไซต์ที่รวบรวมผลงานของสมาชิกแต่ละคนในกลุ่ม จัดทำผ่านภาษา HTML CSS และ Javascript ซึ่งเป็นงานโปรเจคของรายวิชา Web Development I (CMM240) โดยสมาชิกแต่ละคนจะนำผลงานด้านต่างๆของตนเอง มารวบรวมไว้ในเว็บไซต์ ไม่ว่าจะเป็นผลงานด้านการ Design, Drawing, Video รวมถึงผลงานด้านโปรแกรมต่างๆ ทั้งผลงานที่ทำผ่านโปรแกรม Adobe Photoshop, Adobe Indesign, Adobe Illustrator, Adobe After Effects และโปรแกรมอื่นๆอีกมากมาย โดยแต่ละผลงานล้วนมาจากฝีมือและประสบการณ์ของสมาชิกแต่ละคนทั้งหมด

Mhub Website

เว็บไซต์ Mhub เป็นเว็บไซต์ที่ผมฝึกทำโดยใช้ Vue JS ครั้งแรก โดยเว็บไซต์จะเป็นเว็บที่แสดงภาพยนต์ที่น่าสนใจ มีการแสดงชื่อภาพยนต์ เรตติ้ง ประเภท และเรื่องย่อ อีกทั้งยังแสดง Review หนังจากคอมเมนท์ของนักวิจารณ์ รวมถึงบนสื่อออนไลน์ โดยข้อมูลทั้งหมดที่นำมาแสดง จะใช้ Rest API ของเว็บ IMDB มาใช้ทั้งหมด จึงมีความน่าเชื่อถือและถูกต้อง เว็บไซต์นี้มีการแสดงผลแบบ Responsive ที่รองรับทุกขนาดหน้าจออีกด้วย
